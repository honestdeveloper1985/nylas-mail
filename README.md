# K2 - Sync Engine Experiment

# Initial Setup

1. Download https://toolbelt.heroku.com/

2. Install redis for local dev:

```
brew install redis
```

3. Make sure you're using the latest node:

```
nvm install 6
nvm use 6
```

4. `npm install`

# Running locally

```
foreman start -f Procfile.dev
```
